# Token Swap Project

This project demonstrate the token swap functionality from ETH to SMT Token.

It was built using:

Smart Contract:
- Hardhat
- Solidity 0.8.2
- Unit Testing using Waffle and Chai

Front End
- React
- Web3-React for Smart Contract Connection
- BootStrap
- Context for state management.
- Hooks

Deployment:
- Ropsten Network
